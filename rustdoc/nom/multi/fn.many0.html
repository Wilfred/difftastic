Expand description

Repeats the embedded parser, gathering the results in a Vec.

This stops on Err::Error and returns the results that were accumulated. To instead chain an error up, see cut.

ยงArguments

  • f The parser to apply.

Note: if the parser passed in accepts empty inputs (like alpha0 or digit0), many0 will return an error, to prevent going into an infinite loop

use nom::multi::many0;
use nom::bytes::complete::tag;

fn parser(s: &str) -> IResult<&str, Vec<&str>> {
  many0(tag("abc"))(s)
}

assert_eq!(parser("abcabc"), Ok(("", vec!["abc", "abc"])));
assert_eq!(parser("abc123"), Ok(("123", vec!["abc"])));
assert_eq!(parser("123123"), Ok(("123123", vec![])));
assert_eq!(parser(""), Ok(("", vec![])));
