pub struct KeyEventState(/* private fields */);
Expand description

Represents extra state about the key event.

Note: This state can only be read if KeyboardEnhancementFlags::DISAMBIGUATE_ESCAPE_CODES has been enabled with PushKeyboardEnhancementFlags.
