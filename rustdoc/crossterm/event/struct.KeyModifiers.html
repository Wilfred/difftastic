pub struct KeyModifiers(_);
Expand description

Represents key modifiers (shift, control, alt, etc.).

Note: SUPER, HYPER, and META can only be read if KeyboardEnhancementFlags::DISAMBIGUATE_ESCAPE_CODES has been enabled with PushKeyboardEnhancementFlags.
