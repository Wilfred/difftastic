Struct tree_sitter::QueryCaptures

source · 
pub struct QueryCaptures<'query, 'tree: 'query, T: TextProvider<I>, I: AsRef<[u8]>> { /* private fields */ }
Expand description

A sequence of QueryCaptures associated with a given QueryCursor.

During iteration, each element contains a QueryMatch and index. The index can be used to access the new capture inside of the QueryMatch::captures’s [captures].
