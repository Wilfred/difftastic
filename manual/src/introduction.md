# Introduction

Difftastic ([source](https://github.com/wilfred/difftastic)) is a diff
tool that understands syntax. It currently supports 18 languages:

* C
* C++
* C#
* Clojure
* CSS
* Elixir
* Emacs Lisp
* Go
* Haskell
* Java
* JavaScript (and JSX)
* JSON
* OCaml
* Python
* Rust
* TypeScript (and TSX)

When it works, it's *fantastic*.
